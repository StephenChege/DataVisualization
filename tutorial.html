Data Visualization Tutorial Using Seaborn

In order to plot data we first have to set up the plotting environment, select a dataset, load the data set and finally examine the data set

#Set up the environment import pandas as pd import seaborn as sns import matplotlib.pyplot as plt

After setting up the environment we now have to select and load the dataset

#select dataset filepath = "filename.csv" #load dataset #The index_col is used to set the index column or the first column. filename_data = pd.read_csv(filepath, index_col='index')

In order to examine the dataset we use may use the codes below

#Examine the dataset #To prints the first five rows of data; filename_data.head() or # To print a specified number of rows we use n = no.of rows; filename_data.head(n=7) or # To print the last five rows; filename_data.tail() or #To print a specific number of last rows use n = no. of rows; filename_data.tail(n=8) # To prints the whole dataset if its small; filename_data

Linecharts

To plot data using lineplot we use the code

sns.lineplot(data=filename_data)

To improve the graph appearance,

#Set figure width and height respectively, #(the size is in inches) plt.figure(figsize=(5,5)) #Add title, plt.title('Title') #To plot the figure; sns.lineplot(data=filename_data)

We print all columns using the line of code below;

list(filename_data.columns)

To plot a subset of data we use the name of the column we want to plot;

sns.lineplot(data=filename_data['column_name'], label='column name') sns.lineplot(data=filename_data['different_column_name'], label='different column name')

The code block below is a summary of the lines of code to make a good lineplot;

plt.figure(figsize=(10, 7)) plt.title('Title') plt.xlabel('Xlabel') plt.ylabel('Ylabel') #For the whole dataset sns.lineplot(data=filename_data) #For a subset sns.lineplot(data=filename_data['column_name'], label="column name")

Bar charts and Heatmaps

Barplot

To plot a barplot, the code below is used and manipulated to grant the users wishes;

sns.barplot(x=filename_data.index, y=filename_data['column_name'])

Heatmaps

For a heatmap the code below is used. The annot= True displays the numbers in the heatmap. If annot is set to False then the numbers will not be displayed in the heatmap.

sns.heatmap(data=filename_data, annot=True)

Scatter Plots

Simple scatter plot

For a simple scatterplot, we use the code below

sns.scatterplot(x=filename_data['col_name'], y=filename_data['column_name'])

Regression line

To add a regression line to the scatter plot we use the code below

sns.regplot(x=filename_data['col_name'], y=filename_data['column_name'])

Color-coded scatter plots

They display relationship between three variables. An example of the code is shown below

sns.scatterplot(x=filename_data['col_name'], y=filename_data['column_name'], hue=filename_data['column'])

To have two regression lines for a scatter plot use the code below

sns.lmplot(x='col_name', y='column_name', hue='another_column', data=filename_data)

Categorical scatter plot

Categorical data is plotted using the command below

sns.swarmplot(x=filename_data['col_name'], y=filename_data['column_name'])

Distribution Plots

Histograms

To plot a histogram the command below is used. kde has to be provided with its value as false in order to create a histogram.

sns.distplot(a=filename_data['col_name'], kde=False)

Density Plots

Kernel density estimate(KDE) can be thought of as smoothened histograms. The code below is used to produce a kde plot. Setting the shade to true colors the region below the curve.

sns.kdeplot(data=filename_data['col_name'], shade=True)

2D KDE plot

A 2D kde plot can be created using the line of code below

sns.joinplot(x=filename_data['col_name'], y=filename_data['column_data'], kind="kde")

Combined Histogram

To plot a combined histogram, load all data sets you want to plot

sns.distplot(a=filename_data['column_name'], label='column_label',kde=False) sns.distplot(a=filename_data['different_column_name'], label='diff_label',kde=False) sns.distplot(a=filename_data['new_column_name'], label='new_label',kde=False)
#To force the legend to appear,
plt.legend()

Combined Kdeplot

To plot a combined histogram, load all data sets you want to plot

sns.kdeplot(data=filename_data['column_name'], label='column_label', shade=True) sns.kdeplot(data=filename_data['different_column_name'], label='diff_label', shade=True) sns.kdeplot(data=filename_data['new_column_name'], label='new_label', shade=True)

Styling the graph

To style the graph, we use the command below

sns.setstyle("desired style")

The desired style in seaborn has the following five different values

  • darkgrid
  • whitegrid
  • dark
  • white
  • ticks
